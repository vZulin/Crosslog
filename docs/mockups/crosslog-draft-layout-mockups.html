Add Pane Placement Options

Four safer placements for a larger Add Log Pane target, separated from close, search, and file navigation controls.

Option A: TopbarRecommended

Best target safety. It is global, isolated, and works for keyboard shortcuts.

Option B: Header EndcapBrowser-like

Close to the rightmost pane, but only consumes header-height space.

Option C: Floating End RailLarge

Largest hit target. It floats near the pane edge and can appear on hover.

Option D: Splitter ButtonLocal

Adds near the split point. Good for precise pane insertion, less discoverable.
