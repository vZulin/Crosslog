1. В десктоп приложении цветовая схема для темной темы не совпадает с мокапами
Шаги воспроизведения
1. Открыть десктоп приложение в темной теме
2. Сравнить цвета интерфейса с мокапами docs/mockups/crosslog-macos-redesign-mockups.html#dark-mode
Актуальный результат
цветовая схема в приложении отличается от мокапов
Ожидаемый результат
цветовая схема для темной темы в десктоп приложении совпадает с мокапами

2. В десктоп приложении не открывается файловый менеджер для добавления логов и директорий при нажатии на add pane open source
Шаги воспроизведения
1. Открыть десктоп приложение
2. Нажать add pane на crosslog-topbar
Актуальный результат
файловый менеджер не открывается
Ожидаемый результат
открывается системный файловый менеджер, пользователь может выбрать лог или директорию для открытия. если пользователь выбирает файл, открывается File Source, если пользователь выбирает директорию открывается Directory Source 

3. В десктоп приложении не открывается файловый менеджер для добавления логов и директорий при нажатии на open source
Шаги воспроизведения
1. Открыть десктоп приложение
2. Закрыть все Log Pane что бы попасть на empty-workspace
2. Нажать open source
Актуальный результат
файловый менеджер не открывается
Ожидаемый результат
открывается системный файловый менеджер, пользователь может выбрать лог или директорию для открытия. если пользователь выбирает файл, открывается File Source, если пользователь выбирает директорию открывается Directory Source 

4. В десктоп приложении не работает drag&drop для открытия Log Pane
Шаги воспроизведения
1. Открыть десктоп приложение
2. Закрыть все Log Pane что бы попасть на empty-workspace
3. Перетащить файл или директорию в окно приложения
Актуальный результат
ничего не происходит
Ожидаемый результат
перетащенный файл или директория открывается в Log Pane, если пользователь выбирает файл, открывается File Source, если пользователь выбирает директорию открывается Directory Source

5. В веб приложении нельзя открыть директорию, только файл
Шаги воспроизведения
1. Открыть веб приложение
2. Попробовать открыть директорию через add pane на crosslog-topbar или open source на empty-workspace
Актуальный результат
можно открыть только файл
Ожидаемый результат
пользователь может открыть как файл, так и директорию

8. Многие иконки плохо отцентрованы относительно их зоны, которая выделяется при наведении курсора
Шаги воспроизведения
1. Навести курсор на иконки crosslog-icon-button crosslog-sync-toggle, add pane, закрытия pane на crosslog-topbar
2. Навести курсор на activity-rail-search, activity-rail-filter, activity-rail-palette, activity-rail-files, activity-rail-bookmark, activity-rail-settings на  activity-rail
3. Навести курсор на pane-search-next и pane-search-previous на pane-search-popover
4. Навести курсор на Close Settings на settings-surface
5. Проверить кнопки переключения файлов на crosslog-topbar для директории 
Актуальный результат
иконки визуально смещены относительно центра зоны выделения
Ожидаемый результат
все иконки отцентрованы относительно зоны, которая выделяется при наведении курсора

10. crosslog-log-viewport__line-number слишком широкий
Шаги воспроизведения
1. Открыть любой pane с логом
2. проверить ширену crosslog-log-viewport__line-number
Актуальный результат
ширена crosslog-log-viewport__line-number зависит от количества строк в файле, но сильно шире чем нужно
Ожидаемый результат
ширена crosslog-log-viewport__line-number зависит от количества строк в файле, если количество строк 10-99, то ширина отступа должна быть на 2 символа, если 100-999, то на 3 символа, 1000-9999, то на 4 символа и так далее
