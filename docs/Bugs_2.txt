1. В десктоп приложении цветовая схема для темной темы не совпадает с мокапами
Шаги воспроизведения
1. Открыть десктоп приложение в темной теме
2. Сравнить цвета интерфейса с мокапами
Актуальный результат
цветовая схема в приложении отличается от мокапов
Ожидаемый результат
цветовая схема для темной темы в десктоп приложении совпадает с мокапами

2. В десктоп приложении не открывается файловый менеджер для добавления логов и директорий при нажатии на add pane open source
Шаги воспроизведения
1. Открыть десктоп приложение
2. Нажать add pane open source
Актуальный результат
файловый менеджер не открывается
Ожидаемый результат
открывается файловый менеджер, пользователь может выбрать лог или директорию для открытия

3. В десктоп приложении не работает drag&drop для открытия
Шаги воспроизведения
1. Открыть десктоп приложение
2. Перетащить файл или директорию в окно приложения
Актуальный результат
ничего не происходит
Ожидаемый результат
перетащенный файл или директория открывается в приложении

4. В веб приложении нельзя открыть директорию, только файл
Шаги воспроизведения
1. Открыть веб приложение
2. Попробовать открыть директорию
Актуальный результат
можно открыть только файл
Ожидаемый результат
пользователь может открыть как файл, так и директорию

5. При скроле скролится только выбранная позиция, но текст не меняется
Шаги воспроизведения
1. Открыть любой pane с логом
2. Выполнить вертикальный скрол
Актуальный результат
скролится только выбранная позиция, но текст остается на месте
Ожидаемый результат
при скроле текст смещается как в обычных редакторах или браузере

6. Для перемещения pane была добавлена кнопка, за которую нужно тянуть, но нужно чтобы была возможность начать за любое место pane header
Шаги воспроизведения
1. Открыть несколько pane
2. Попробовать начать перемещение pane не за специальную кнопку, а за другое место pane header
Актуальный результат
перемещение pane начинается только при перетаскивании за специальную кнопку
Ожидаемый результат
перемещение pane можно начать за любое место pane header

7. Многие иконки плохо отцентрованы относительно их зоны, которая выделяется при наведении курсора
Шаги воспроизведения
1. Навести курсор на иконки синхронизации скрола, add pane, закрытия pane, иконки на боковой панели и стрелки в search popover
Актуальный результат
иконки визуально смещены относительно зоны выделения
Ожидаемый результат
все иконки отцентрованы относительно зоны, которая выделяется при наведении курсора
